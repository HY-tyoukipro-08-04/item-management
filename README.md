# 商品管理システム

## 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことが出来ます。

商品の新規登録から編集、削除、検索を行うことができ、各商品の在庫を調整することが出来ます。

## 主な機能
- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録、編集、削除機能
- 商品在庫管理機能
- 商品検索機能

## 開発環境
- PHP 8.2.0
- MySQL 5.2.0
- Laravel 10.13.5
